📚 MUNDO DE LAS ANUALIDADES

Aprende las fórmulas mágicas de Excel para dominar las anualidades.

🔵 =VA( ) — VALOR ACTUAL

Calcula el valor HOY de una serie de pagos futuros.
Responde: "¿Cuánto vale ahora un préstamo?"

=VA(tasa; nper; pago; [vf]; [tipo])

📌 El resultado es NEGATIVO porque representa una salida de dinero.

🟢 =VF( ) — VALOR FUTURO

Calcula cuánto valdrá una inversión en el FUTURO.
Responde: "¿Cuánto acumularé si ahorro X por periodo?"

=VF(tasa; nper; pago; [va]; [tipo])

📌 El resultado es POSITIVO — es lo que recibirás en el futuro.

🔴 =PAGO( ) — RENTA PERIÓDICA

Calcula el pago periódico necesario para saldar un préstamo o alcanzar una meta.
Responde: "¿Cuánto pago al mes?"

=PAGO(tasa; nper; va; [vf]; [tipo])

📌 El resultado es NEGATIVO — indica lo que debes pagar.

⚙️ EL ARGUMENTO [TIPO]
